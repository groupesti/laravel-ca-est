# Support

## Getting Help

- **Bugs**: Open a [GitHub Issue](https://github.com/groupesti/laravel-ca-est/issues) with a minimal reproduction case. Include your PHP version (8.4+), Laravel version (12.x or 13.x), and package version.
- **Questions**: Use [GitHub Discussions](https://github.com/groupesti/laravel-ca-est/discussions) for general questions, usage help, and feature ideas.
- **Security**: See [SECURITY.md](SECURITY.md) for reporting vulnerabilities. Do **not** use GitHub Issues for security reports.

## Before Opening an Issue

1. Search existing issues and discussions to avoid duplicates.
2. Verify you are running PHP 8.4+ and Laravel 12.x or 13.x.
3. Ensure the package is up to date: `composer update groupesti/laravel-ca-est`.
4. Include the relevant configuration from `config/ca-est.php` (with secrets redacted).

## GitHub Issues Are Not a Support Forum

Issues are reserved for **confirmed bugs** and **actionable feature requests**. General usage questions should go to GitHub Discussions.
